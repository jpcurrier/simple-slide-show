# simple-slide-show
Basic jQuery plugin

## Basic Usage
```
<div class="simple-slide-show loading" data-speed="5000">
  <ul class="slides">
    <li class="on">
      <figure>
        <p>First Slide</p>
      </figure>
    </li>
    <li>
      <figure>
        <p>Second Slide</p>
      </figure>
    </li>
  </ul>
</div>
```
